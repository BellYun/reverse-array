# 문제

> 숫자로 이루어진 배열을 입력받아 숫자를 뒤집어 배열로 만드세요.

# 입력

- 인자 1 : arr
  - `number` 타입을 요소로 갖는 배열
  - arr[i]는 0 이상의 정수

# 출력

- `number` 타입을 요소로 갖는 배열을 리턴해야 합니다.
- arr[0]은 arr[len - 1], arr[1]은 arr[len - 2], ..., arr[i]는 arr[len - 1 - i]로 바뀌어야 합니다.

# 준비

## package 설치

```bash
$ npm install
```
